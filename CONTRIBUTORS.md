## Getting Started as a Contributor

**New to contributing?** Start here:

1. Fork the repository
2. Pick a small issue to start with
3. Follow [CONTRIBUTING.md](CONTRIBUTING.md)
4. Ask questions if stuck
5. Submit your first PR
6. Your name appears in contributors!

**Find issues labeled:**
- `good first issue` - Great for newcomers
- `help wanted` - Need community input
- `documentation` - Help improve docs

---

## Contributor Levels

### Level 1 - Getting Started
- Report bugs
- Fix typos
- Improve documentation
- Answer questions in issues

### Level 2 - Regular Contributor
- Submit features or fixes
- Participate in code reviews
- Help other contributors
- Write tests

### Level 3 - Maintainer
- Lead feature development
- Review pull requests regularly
- Manage releases
- Make architectural decisions
